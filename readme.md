//how to set up server

1. Download Node.js

2. npm install

3. using "npm init"  to create package.json

4. install express

5. set up route and app.js

//how to load web app
1. use cmd locate to the web app file
2. use the code "node app.js"
3. open browser input "localhost:3000"
